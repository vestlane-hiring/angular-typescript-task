# Playwright E2E tests

This tests are based on the [Playwright](https://playwright.dev/) E2E framework.

## How to run it
Install dependencies
```
$ npm install
```
Install chromium dependencies for Playwright
```
$ npx playwright install --with-deps chromium
```
Execute E2E test
```
$ npm run e2e
```
E2E Tests report
```
$ npm run report
```
